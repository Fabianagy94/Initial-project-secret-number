# Initial Project: Secret Number 🎯

Este es mi primer proyecto web interactivo, desarrollado como parte del programa **Oracle Next Education** en colaboración con **Alura Latam**. El juego consiste en adivinar un número secreto generado aleatoriamente, con pistas que guían al jugador hasta encontrar la respuesta correcta.

This is my first interactive web project, developed as part of the **Oracle Next Education** program in collaboration with **Alura Latam**. The game challenges players to guess a randomly generated secret number, with clues that guide them toward the correct answer.
